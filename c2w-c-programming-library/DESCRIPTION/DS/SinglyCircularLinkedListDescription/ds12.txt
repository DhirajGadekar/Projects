


    SELECTED METHOD:

        PRINT PALINDROME ELEMENT INDEX IN LINKEDLIST.

    DESCRIPTION:

        This function checks if palindrome number is present in LinkedList or not,
        and return index of occurrence.

    EXAMPLE:

        Input:

            ---->|10|-->|2|-->|30|-->|11|----
            |________________________________|
            

        Output:
                2
                4
