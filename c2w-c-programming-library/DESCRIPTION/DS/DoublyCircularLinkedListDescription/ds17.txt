


    SELECTED METHOD:

        REVERSE LINKEDLIST INPLACE.

    DESCRIPTION:

        This function reverse LinkedList inplace.

    EXAMPLE:

        Input:
               |--><--|10|<-->|20|<-->|30|<-->|40|--><--|
               |________________________________________|


        Output:
           
               |--><--|40|<-->|30|<-->|20|<-->|10|--><--|
               |________________________________________|
