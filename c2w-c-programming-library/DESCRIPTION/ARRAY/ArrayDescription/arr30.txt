


        SELECTED METHOD :   
                
            30. GROUPING OF COMPOSITE NUMBER

        DESCRIPTION :   
            
            A composite number is a natural number or a positive integer            
            which has more than two factors.            
            For example, 15 has factors 1, 3, 5 and 15   

            This function takes array address and array length as input 
            and group Composite elements in array.  

        EXAMPLE :   
            
            INPUT ARRAY :  
        
                | 11 | 12 | 13 | 14 | 15 | 16 | 17 | 18 | 19 |    
        
            OUTPUT ARRAY :   
        
                | 12 | 14 | 15 | 16 | 18 |
